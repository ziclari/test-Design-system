    box

    @mixin box($width, $height) { ... }

    Description

    Genera un contenedor con un ancho y alto específico. Si no se proporciona un valor para el height, se usa el valor de width como predeterminado.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $width

    El ancho del contenedor.

    		Length— none
    $height

    La altura del contenedor.

    		Length— none

    Example

    // Usando ambos parámetros (ancho y alto)
.container {
  @include box(200px, 300px);
}

// Usando solo el ancho, el alto se ajustará al valor del ancho
.container {
  @include box(200px);
}

    font-size

    @mixin font-size($size, $line-height, $letter-spacing) { ... }

    Description

    Mixin para definir el tamaño de fuente, altura de línea y espaciado de letras, basado en la escala de tamaños definida.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $size

    Tamaño de fuente de la escala. Ejemplo: 300 para tamaño base.

    		Number— none
    $line-height

    Altura de línea (opcional, por defecto es normal).

    		Number— none
    $letter-spacing

    Espaciado entre letras (opcional, por defecto es normal).

  // font-size: 0.8rem;
  // line-height: 1.5;
  // letter-spacing: 0.1em;
}

    Description

    Mixin para aplicar propiedades de posicionamiento CSS. Este mixin aplica la propiedad position y acepta argumentos para top, right, bottom, y left si se proporcionan. Si no se pasan valores específicos para top, right, bottom, o left, no se aplican.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $position

    El tipo de posicionamiento (absolute, fixed, relative).

    		String— none
    $args

    Los valores opcionales de top, right, bottom, left.

    		List— none

    Example

    .element {
  @include position(absolute, top 10px left 20px);
  // Esto aplicará: position: absolute; top: 10px; left: 20px;
}

    Used by

    • [mixin] absolute
    • [mixin] fixed

    responsive-ratio

    @mixin responsive-ratio($x, $y) { ... }

    Description

    Mixin para mantener la relación de aspecto de un elemento (generalmente imágenes o fondos). Este mixin se utiliza para crear elementos escalables que mantienen una proporción específica. La proporción se define mediante dos valores: el ancho ($x) y el alto ($y). Si $pseudo es true, se usará un pseudo-elemento &:before para mantener la proporción en lugar de aplicarlo directamente al elemento.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $x

    El valor del ancho (proporción de ancho).

    		Number— none
    $y

    El valor del alto (proporción de alto).

    		Number— none

    Example

    div {
  @include responsive-ratio(16, 9);
  // Esto hará que el div tenga una relación de aspecto 16:9.
}

    responsive-font-size

    @mixin responsive-font-size($min-size, $max-size, $min-vw, $max-vw) { ... }

    Description

    Mixin para aplicar un tamaño de fuente responsivo basado en el ancho de la pantalla (viewport). Usa la función clamp() para establecer un rango entre un tamaño de fuente mínimo y máximo, ajustándose dinámicamente entre los valores dependiendo del ancho del viewport.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $min-size

    El tamaño mínimo de la fuente (por ejemplo, '12px').

    		String— none
    $max-size

    El tamaño máximo de la fuente (por ejemplo, '24px').

    		String— none
    $min-vw

    El valor mínimo del ancho de la pantalla (en vw, por ejemplo '320px').

    		String— none
    $max-vw

    El valor máximo del ancho de la pantalla (en vw, por ejemplo '1440px').

    		String— none
